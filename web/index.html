Voice & Text · OpenClaw Client

A minimal iPhone client for talking to your OpenClaw agents — by voice or by text. Full conversation context maintained.

v0.1 · simple and made with care · no accounts · no data collection

Apple Speech for STT, server-side TTS — voice is enhancement, not requirement
Self-hosted OpenClaw backend via OpenAI-compatible API
No credentials stored on device — server URL only
iOS 17+ — requires your own OpenClaw instance
No data collected. No account. Conversations stay on your server.
